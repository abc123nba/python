## 算术运算符

- +	加 - 两个对象相加	 
- -	减 - 得到负数或是一个数减去另一个数	 
- *	乘 - 两个数相乘或是返回一个被重复若干次的字符串	 
- /	除 - x除以y	 
- %	取模 - 返回除法的余数	 
- **	幂 - 返回x的y次幂	a**b 为10的20次方， 输出结果 100000000000000000000
- //	取整除 - 返回商的整数部分（向下取整）
