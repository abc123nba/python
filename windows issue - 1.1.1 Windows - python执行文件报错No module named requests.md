# Windows - python 执行文件报错 No module named requests

>没有安装 requests 模块， 

可以切换到 python 的安装目录找到 script 文件夹 

进入cmd窗口切换到上面的目录直接运营下面两个命令中的一个 
```
1. > Path\pip install requests 
2. > Path\easy_install.exe requests 
```


pip list 显示安装
